Device

Device

A CANopen device.

This class represents a single addressable device (or node) on the bus and provides methods for manipulating the object dictionary.

Constructor

new Device(args)

Source:
  • device.js, line 48
Example
const can = require('socketcan');

const channel = can.createRawChannel('can0');
const device = new Device({ id: 0xa });

channel.addListener('onMessage', (message) => device.receive(message));
device.setTransmitFunction((message) => channel.send(message));

device.init();
channel.start();
Parameters:
Name Type Description
args object
Properties
Name Type Description
id number

device identifier [1-127].

the device's electronic data sheet.
loopback boolean

enable loopback mode.
Fires:
  • event:'message' on receiving a CAN message.
  • event:'emergency' on consuming an emergency object.
  • event:'nmtTimeout' on missing a tracked heartbeat.
  • event:'nmtChangeState' on change of NMT state.
  • event:'nmtResetNode' on NMT reset node.
  • event:'nmtResetCommunication' on NMT reset communication.
  • event:'sync' on consuming a synchronization object.
  • event:'time' on consuming a time stamp object.
  • event:'pdo' on updating a mapped pdo object.
