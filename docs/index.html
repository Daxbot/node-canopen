Pdo

CANopen PDO protocol handler.

The process data object (PDO) protocol follows a producer-consumer structure where one device broadcasts data that can be consumed by any device on the network. Unlike the SDO protocol, PDO transfers are performed with no protocol overhead.

Nmt

CANopen NMT protocol handler.

The network management (NMT) protocol follows a master-slave structure where NMT objects are used to initialze, start, monitor, reset, or stop nodes. All CANopen devices are considered NMT slaves with one device fulfilling the role of NMT master.

This class implements the NMT node control services and tracks the device's current NMT slave state.
